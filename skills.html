Scrapify : Scraping Project

Presentation

Scrapify is of all my personal projects.

It's a little python software allowing you to automate some tasks. Indeed, this tool allow to automaticly get movies from their names and it opens a web tab with the movie in it. It also allows you to get download URLs of console games.

The software has an in-shell menu allowing to choose which function you want to use from it.

This project allowed me to discover the "scraping" (Automatic gathering of informations from websites) universe and can now make this type of programs with the help of the "BeautifulSoup" library and "requests".

It's not my only project thata I realised in python and some of them still being available on my github (It is possible that they're outdated because of the upgrades of the libraries).
